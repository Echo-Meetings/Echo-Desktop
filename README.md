**Private, offline transcription app for macOS.**

Transcribe audio and video files entirely on your Mac using [WhisperKit](https://github.com/argmaxinc/WhisperKit). No internet required, no data ever leaves your device.

**Приватное офлайн-приложение для транскрибации на macOS.**

Транскрибируйте аудио и видео файлы полностью на вашем Mac с помощью [WhisperKit](https://github.com/argmaxinc/WhisperKit). Без интернета, данные никуда не отправляются.

---

## Features / Возможности

- **100% local processing** — all transcription happens on-device using WhisperKit
- **Audio & video support** — MP3, WAV, M4A, MP4, MOV, and more
- **Synchronized playback** — click any transcript line to jump to that moment in the video
- **History & search** — sidebar with full transcription history, search, and date filters
- **Export** — transcripts saved as .txt files
- **Privacy-first** — no accounts, no cloud, no tracking

## Installation / Установка

1. Download `Echo-1.0.0.dmg` from [Releases](../../releases/latest)
2. Open the DMG and drag Echo to Applications
3. Launch Echo — the voice model downloads automatically on first run (~1-3 GB, one time only)

## Requirements / Требования

- macOS 14 (Sonoma) or later
- Apple Silicon (M1 / M2 / M3 / M4)
